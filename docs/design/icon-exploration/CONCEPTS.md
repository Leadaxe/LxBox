# LxBox Icon — Concept Exploration

## Brand anchors
- **Name**: L×Box (L «умножить на» Box). × — не просто буква: это **умножение/пересечение/крестик**.
- **Продукт**: Android VPN клиент на sing-box. Мульти-подписки, умный routing, speed-test.
- **Смыслы**: маршрутизация, защита, скорость, множество каналов → один ящик, «свобода через коробку».
- **Аудитория**: RU/CN tech-savvy, power users, разработчики, конфиденциальность.
- **Тональность**: технологичная, точная, не «vpn-щит-зелёный», не инфантильная.

## Referenced visual directions (moodboard pointers)
- **Sing-box / Clash Meta / NekoBox** — минималистичные коробки/иероглиф в круге, flat-tech.
- **Outline Manager, Mullvad, IVPN** — геометрические минималистичные марки без «защитного щита».
- **Docker, Cloudflare Warp, Tailscale** — «контейнер»/«сеть»/«граф» как язык формы.
- **Material You expressive** — мягкая геометрия, глубокие градиенты, тёплая акцентная палитра.
- **Asian indie-utility** — NekoRay, Hiddify, v2rayN — 2D-символ + буква, высокий контраст.
- **Retro CRT / terminal-core** — phosphor-green, scanlines, моно-цифры. Актуален у прокси-утилит.
- **Isometric box / 3D-parcel** — куб в 3/4 перспективе, рёбра-«провода».

## 25 концепций

### Блок A — «Коробка как контейнер»
1. **Isometric cube + L-шнур** — изометрический куб (box), из него выходит «L-образный» tunnel-кабель, загибающийся вверх. Читается как «вывожу трафик из коробки».
2. **Open-top box с молниями внутри** — вид сверху на открытую коробку, внутри пучок светящихся «молний» (каналы-подписки).
3. **Box с QR-узором на грани** — куб, грань — стилизованный QR/mesh, намёк на подписки/ноды.
4. **Packet-box** — куб, составленный из шести сетевых «пакетов» (квадратиков-пикселей), зазоры между гранями.
5. **Collapsed box (⊞/⊟)** — «⊞» как символ: квадрат с крестом внутри. Буквально logo-type.

### Блок B — «× как смысл»
6. **L×** как монограмма — две буквы в круге, × — крупнее, «умножает» L. Цветной × выступает за контур L.
7. **Routing-cross** — × состоит из двух перекрещивающихся траекторий-стрелок (одна сверху-справа вниз-влево, другая обратная), сходятся в точке-узле — это и есть «маршрутизация».
8. **× = шифр** — крестик из двух ключей (key-glyph), перекрещенных, concealed за «L».
9. **Node-cross** — × как граф из 4 узлов и 4 рёбер, в центре — bright node (auto-proxy-out).
10. **Split-gate ×** — × как разделитель потоков: трафик входит слева, расходится на две ветки (direct/proxy) — напрямую отражает фичу routing.

### Блок C — «Маршрутизация / сеть»
11. **Tri-path junction** — три стрелки разной длины, сходящиеся в одну точку (три VPN группы — ①②③ из бренда).
12. **Subway-map** — три цветные линии (как метро), каждая — подписка, пересекаются в «станции» L×Box.
13. **Constellation box** — грань куба как созвездие из нод, одна нода «горит» — активная.
14. **Flow diode** — стрелка, проходящая через квадратный «фильтр» (box), на выходе — очищенный/зелёный поток.
15. **Switchboard** — ретро-коммутатор, три кабеля воткнуты в три порта (подписки → один VPN).

### Блок D — «Защита без штампов»
16. **Soft-shield box** — НЕ классический щит. Куб с лёгким «ореолом/halo» — защитное свечение вокруг.
17. **Chamber** — куб как «камера»/шлюз, внутри силуэт L, снаружи — опасный хаос линий, внутри — порядок.
18. **Closed-open duality** — половина иконки — замочек, вторая — открытый поток. Privacy + freedom.
19. **Vault-L** — L-образная прорезь в монолитной плите-сейфе, сквозь неё льётся цвет.
20. **Aperture-box** — объектив/apertura, лепестки складываются в квадрат. Контроль видимости.

### Блок E — «Tech/dev/retro»
21. **Terminal L×** — моно-шрифт `L×` + мигающий курсор, на фоне soft CRT-glow. Для dev-публики.
22. **Hex-pill box** — квадрат с закруглением, внутри `0x4C42` (ASCII L B) стилизованно.
23. **Command-box** — квадратная скобка `[` `]` обрамляющая `L×`. Отсылает к sing-box CLI.
24. **Glitch-cube** — изометрический куб с 1-pixel shift одной грани (RGB split), ощущение «proxy/tunnel».
25. **Signal-bars as box** — сигнал-бары, сгруппированные в форму куба. Скорость + контейнер.

### Бонус (в запас)
26. **Origami-L** — сложенный из бумаги L, грани разных оттенков. Мягко, не «vpn».
27. **Möbius-L** — лента Мёбиуса в форме L. Туннель без начала-конца.
28. **Chess-rook box** — ладья = башня = крепость + путь только прямо (routing).
29. **Dice-×** — игральная кость, на верхней грани × из 4 точек.
30. **Portal-L** — два L-образных портала, между ними искажение пространства (туннель).

## Критерии отбора (скоринг 1–5 по каждому)
- **Distinctive** — не путается с конкурентами (1-й экран Play Store).
- **Recognizable @48px** — читаемо как launcher-icon на Android.
- **On-brand** — передаёт routing/multi-sub/box, не «generic VPN».
- **Scalable** — работает монохромом, на тёмном, в adaptive-icon (foreground+background).
- **Taste** — просто красиво, не кринж, не 2014.
- **Executable** — можно нарисовать в SVG без эффектов-костылей.
