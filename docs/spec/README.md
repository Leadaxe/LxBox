# Спецификации (BoxVPN)

Каталог для требований, архитектурных заметок и планов по продукту.

## Структура

| Папка | Назначение |
|-------|------------|
| [`features/`](features/) | Спецификации **фич** — что строим, поведение, критерии приёмки. |
| [`tasks/`](tasks/) | Рабочие **задачи** — декомпозиция, чеклисты, статус; в т.ч. [`004`](tasks/004%20mvp%20implementation/spec.md) (реализация MVP), [`005`](tasks/005%20cicd%20github%20actions/spec.md) (CI/CD). |

Связь: у фичи и связанных задач **один номер** и одно **человекочитаемое имя папки**: `NNN <название с пробелами>/spec.md`, например `features/001 mobile stack/spec.md` и `tasks/001 mobile stack/spec.md`.

## Соглашения

- Имя папки: **префикс номера** (`001`, `002`, …), **пробел**, затем **название из нескольких слов через пробел** (без подчёркиваний в названии).
- Внутри папки основной файл — **`spec.md`**.
- В начале `spec.md` — краткий контекст, цели и нецели (для продуктовых фич).
