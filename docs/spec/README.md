# Спецификации (BoxVPN)

Каталог для требований, архитектурных заметок и планов по продукту.

## Структура

Все спецификации находятся в [`features/`](features/). Каждая фича — отдельная папка.

Внутри папки фичи:

| Файл | Назначение |
|------|------------|
| `spec.md` | Спецификация: что строим, поведение, критерии приёмки |
| `plan.md` | План реализации (при необходимости) |
| `tasks.md` | Задачи: декомпозиция, чеклисты, статус |

## Соглашения

- Имя папки: **префикс номера** (`001`, `002`, …), **пробел**, затем **название из нескольких слов через пробел**.
- Основной файл — **`spec.md`**; дополнительные (`plan.md`, `tasks.md`) — по потребности.
- В начале `spec.md` — краткий контекст, цели и нецели.
