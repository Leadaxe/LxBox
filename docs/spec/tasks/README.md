# Задачи

Здесь ведутся **задачи на реализацию**: декомпозиция работ, порядок, статус, ссылки на PR и на спецификации фич в [`../features/`](../features/).

**Имя папки:** то же, что у фичи — `NNN <название с пробелами>`. Внутри — **`spec.md`**.

Примеры:

- разбивка эпика на подзадачи;
- чеклист перед релизом;
- технический долг с приоритетом.
