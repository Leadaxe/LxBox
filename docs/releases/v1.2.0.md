# L×Box v1.2.0

Release date: 2026-04-18

## Highlights

### Outbound groups overhaul
- Renamed **proxy-out → vpn-1**, added **vpn-3** (VPN ①/②/③ available).
- **VPN ①** is always generated — toggle is locked (required base group).
- **auto-proxy-out** is now controlled by the **Include Auto** toggle:
  - ON → generated as `urltest` and added to `vpn-*` groups.
  - OFF → section is not produced at all, `vpn-*` groups are not referencing it.

### Node list UX
- `direct-out` and `auto-proxy-out` are pinned to the top in every sort mode (order: direct, then auto), with a subtle highlight.
- Long-press context menu:
  - No Copy actions for `direct-out` / `auto-proxy-out`.
  - *Copy detour* and *Copy server + detour* hidden when node has no detour.

### Defaults
- `urltest_tolerance` default 30 ms (was 100).

## Files
- `app-release.apk` — signed release build for Android.
