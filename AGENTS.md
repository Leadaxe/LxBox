# Руководство для агентов (L×Box)

Правила для AI-агентов и автоматизации, работающих с этим репозиторием.

## Git: коммиты и отправка на сервер

**Не выполнять** без **явного указания оператора** (пользователя):

- создание коммитов (`git commit`, слияния с коммитом и т.п.);
- отправка на удалённый репозиторий (`git push`, `git push --force`, публикация веток);
- действия CLI, которые подразумевают пуш (например, `gh pr create` с автоматическим пушем), если пользователь прямо не попросил опубликовать изменения.

Допустимо: правки файлов в рабочей копии, `git status`, `git diff`, подготовка сообщений коммита **в чате без выполнения** — если пользователь сам выполнит коммит.

Исключение: пользователь **явно** просит закоммитить, запушить или синхронизировать с сервером — тогда выполнять в соответствии с запросом.
